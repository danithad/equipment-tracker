# Equipment Tracker

A simple full-stack web application to manage and track equipment records.  
This project demonstrates CRUD operations, REST API integration, and database persistence using modern web technologies.

---

## Features

- View a list of equipment in a table
- Add new equipment
- Edit existing equipment
- Delete equipment
- Select and store Last Cleaned Date using a date picker
- Search equipment by name, type, or status
- Responsive and clean UI

---

## Tech Stack

### Frontend
- React (Vite)
- Lovable-generated UI
- shadcn/ui components
- React Hook Form + Zod (validation)
- React Query (data fetching & caching)
- Axios

### Backend
- Node.js
- Express.js
- RESTful APIs

### Database
- MongoDB Atlas (cloud MongoDB)
